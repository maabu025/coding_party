# React + TypeScript Guided Learning Project

##  Project Overview
This project is a guided learning application built with **React**, **TypeScript**, and **Vite**.  
It is designed to demonstrate both **foundational and advanced concepts** in React component architecture, styling, and state management using TypeScript for type safety.

The project is structured around **three major task categories**, each containing multiple tasks that progressively build proficiency in React + TypeScript development.

---

##  Learning Objectives
By completing this project, the following skills are demonstrated:

- Creating and typing React functional components with TypeScript
- Using props, default props, and optional props safely
- Structuring reusable and nested components
- Styling components using multiple approaches
- Managing local state with `useState` (typed)
- Handling typed events (clicks, forms, inputs)
- Lifting state up and passing state via props
- Applying best practices in project structure and Git workflow



## Technologies Used
